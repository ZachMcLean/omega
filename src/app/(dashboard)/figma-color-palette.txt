üí° FIGMA TIPS:
=============

1. Create a "Design Tokens" page in your Figma file
2. Place all color swatches in organized frames
3. Use the "Styles" feature to save all colors
4. Export styles as a library to use across files
5. Name styles consistently: Category/Variant
6. Use plugins like "Design Tokens" to export/import
7. Create component variants for different states

RECOMMENDED PLUGINS:
- Design Tokens (export to JSON)
- Stark (accessibility checking)
- Color Contrast Checker
- Variables Import/Export

üé® STYLE DESCRIPTION:
====================

"Neon Fintech" or "Cyberpunk Trading Dashboard"

A vibrant, high-energy design combining sleek modern 
fintech (Robinhood-style) with gaming dashboard 
excitement. Dark backgrounds with neon-bright accents, 
extensive gradients, glows, and color-coded data for 
instant visual information scanning.

Key Characteristics:
- Dark slate base (#0F172A - #334155 range)
- Bright saturated accent colors
- Extensive use of opacity overlays
- Gradient backgrounds and fills
- Glowing shadows and effects
- Color = semantic meaning (green=up, red=down, etc.)
- Gaming energy meets professional finance
